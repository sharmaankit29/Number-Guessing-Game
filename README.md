# 🎯 Number Guessing Game

An interactive web-based Number Guessing Game built using HTML, CSS, and JavaScript. The game generates a random number between **1 and 100**, and the player must guess the correct number within **5 attempts**.

## 🚀 Features

* Random number generation between 1 and 100
* Limited number of attempts (5 guesses)
* Input validation for invalid entries
* Real-time feedback:

  * Too High
  * Too Low
  * Correct Guess
* Displays previous guesses
* Shows remaining attempts
* Restart game functionality
* Dynamic UI updates using JavaScript DOM manipulation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* DOM Manipulation

## 📂 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/number-guessing-game.git
```

2. Navigate to the project folder:

```bash
cd number-guessing-game
```

3. Open `index.html` in your browser.

## 🎮 How to Play

1. Enter a number between **1 and 100**.
2. Click on **Submit Guess**.
3. The game will tell you whether your guess is too high or too low.
4. You have **5 attempts** to guess the correct number.
5. Click **Start New Game** to play again.
